# HavenStay 🏨

A modern, high-performance real estate and rental management platform designed to connect verified tenants with asset owners seamlessly. This platform streamlines property listings, admin approval cycles, analytics tracking, and role-based access management.

## Purpose

**HavenStay** is engineered to eliminate the friction from modern property leasing and listing management. The system provides:

- **For Owners:** A fast, programmatic way to list assets, track multi-line media portfolios, and observe transparent monthly earnings tracking.
- **For Tenants:** A clean interface to search, browse, and secure verified real estate accommodations safely.
- **For Admins:** A specialized centralized command station to monitor structural platform metrics, approve/reject property listings with direct feedback streams, and safely execute user authorization and role mutation workflows.

---

## Key Features

### Property Hub & Asset Management

- **Dynamic Asset Form Creation:** Intuitive wizard layouts supporting advanced validations, dynamic structural category allocations (Apartments, Villas, Penthouses, Cabins, Studios), and multi-line image URL handling.
- **Inline Editing Modals:** Fully functional, real-time edit dialog frames allowing asset owners to dynamically modify current records, automatically resetting approval states back to a pending confirmation lifecycle.

### Administrative Console & Control Rooms

- **Live Moderation Architecture:** Quick-action asset evaluation loops featuring one-click status approvals and dynamic rejection pipelines coupled with granular admin feedback injections.
- **Dynamic Identity Mutation System:** Secure user tables utilizing custom select menus to safely switch account capabilities between `Tenant`, `Owner`, and `Admin` classes instantly.

### Real-Time Analytics Panels

- **Interactive Metric Visualizations:** Custom analytical distribution dashboards rendering platform structures, verification streams, and financial records cleanly using responsive graphical layouts.
